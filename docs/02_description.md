# Description

## Abstract

Northern Territory Geological Survey (NTGS) geochemistry datasets contain information on available open-file geochemical data from companies and/or data collected by NTGS on stream sediments.

The data are captured by the department via compulsory reporting from exploration companies or analyses conducted by NTGS geologists.
